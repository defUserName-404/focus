Functional Requirements:
1.    Cross Platform App: should work on mobile (android, ios) and Desktop (mac, windows, linux)
2.    Offline app
3.    Project management: creating tasks, sub tasks, sub- sub tasks etc
4.    Projects should have description, start and deadline
5.    Tasks should also have the description, priority(critical, high, medium, low), start and end time
6.    Tasks should be tracked with Focus sessions. Each focus sessions could be customized with a small break session from the settings.
7.    Each sessions will play some noise/ or no noise(customizable from the settings)
8.    After session ends, the app should play a small notification with an alarm
9.    While the session is active, the app should keep the session playing in the background and display a notification just like a music player
