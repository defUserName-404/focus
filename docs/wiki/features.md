
# Features (Full)

## Project Management
- Create, edit, delete projects
- Progress tracking, metadata, sorting/filtering
- Project detail screens with inline subtask add/edit

## Task & Subtask Management
- Hierarchical tasks and subtasks
- Priority, due dates, sort/filter chips
- Inline add/edit modals, confirmation dialogs

## Focus Sessions
- Start, pause, stop sessions
- Session timer, progress, break/ambience logic
- Marquee animation for session status

## Notifications & Ambience
- Custom notification actions (pause, resume, stop)
- Real-time updates for play position, time
- Ambience mute/unmute, computed provider logic

## UI/UX
- Modular widgets: AppCard, AppSearchBar, modal forms
- Pixel-perfect alignment, responsive design
- Command pattern for actions

## Data Layer
- Drift ORM, schema migrations, indexing
- Efficient querying, watchProjectById, sort/filter at ORM level

## Search, Sort, Filter
- Search bars, filter chips, sort order selectors
- Debounced search, ORM-level filtering

## Platform Support
- Android, iOS, Linux, macOS, Windows, Web

## Accessibility
- Clean separation of concerns, maintainable codebase
