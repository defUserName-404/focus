# Focus App Wiki Structure

This folder contains the modular project wiki for the Focus open source app. Each file covers a specific aspect of the project, following standard open source documentation practices.

## Files
- README.md: Wiki structure and navigation
- overview.md: Project overview and goals
- getting_started.md: Installation and setup
- features.md: Feature list and usage
- architecture.md: Technical design and structure
- contributing.md: Contribution guidelines
- faq.md: Frequently asked questions
- changelog.md: Project changelog

## Adding Screenshots
Add screenshots to this folder or reference them in the relevant markdown files. Use descriptive filenames and update the documentation as needed.

## Release Notes
Release notes can be added to changelog.md or as separate files for each release.
